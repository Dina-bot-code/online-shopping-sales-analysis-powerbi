# 📊 Online Shopping Sales Analysis Dashboard

## Overview

Developed an interactive Power BI dashboard to analyse e-commerce sales performance, customer purchasing behaviour, and revenue trends using online shopping transaction data.

## Tools Used

- Microsoft Excel
- Data Cleaning
- Data Visualisation
- Business Intelligence
